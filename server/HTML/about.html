About Us

How it started?

This website had been built by Divya Patel, Bhumika Devaliya, Hemil Darji, Dev Solanki and Yash Panchal to solve the problem of all people from all ages but especially for old aged person who have to take so many of medicine on everyday basis because of some kind of diseases or anything that they might suffering from, so due to this we make this website to remind them for taking their medicine on time without forgetting it.

Functionality

This website helps people for intaking their medicine on time without forgetting it, but yes they should have to be logged on in our website, so that whatever Email they had use at the time of login they will get their notification on that particular mail-id so they have to be active on that. Additionally, they independently set the timer of as many medicine they want to take and also they have to enter their drugs name so that they get to know which medicine they have to take with mentioning their pricing for particular tablet or any medicine to keep the record of it, so they get to know how much they spent on it.

How it's made?

We use some basic tools to made this webiste like Visual Studio Code. So, Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js and has a rich ecosystem of extensions for other languages (such as C++, C#, Java, Python, PHP, Go) and runtimes (such as .NET and Unity). Moreover, we use MongoDB for storing the user data in it. MongoDB is a document-oriented NoSQL database used for high volume data storage. Instead of using tables and rows as in the traditional relational databases, MongoDB makes use of collections and documents. Documents consist of key-value pairs which are the basic unit of data in MongoDB. And some other languages for making our webiste efficienct.
